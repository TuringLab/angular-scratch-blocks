#Angular Scratchblocks

Angular scratchblocks is a simple angular interface for [scratchblocks2](https://github.com/tjvr/scratchblocks)

##Usage

Angular Scratchblocks can be installed through bower with the following command

```
bower install angular-scratchblocks
``` 

##Development

Please feel free to improve this limited respository

- clone the repo
- install node dependencies
- install bower dependencies

```
npm install
bower install
```
